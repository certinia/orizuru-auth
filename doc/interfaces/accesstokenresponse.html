A URL indicating the instance of the user’s org. For example: https://yourInstance.salesforce.com/.

issued_at

issued_at: string

When the signature was created.

refresh_token

refresh_token: string
  • Defined in flow/response/accessToken.ts:71

Token that can be used in the future to obtain new access tokens (sessions). This value is a secret. Treat it like the user’s password, and use appropriate measures to protect it. This parameter is returned only if your connected app is set up with a scope of at least refresh_token.

signature

signature: string
  • Defined in flow/response/accessToken.ts:98

Base64-encoded HMAC-SHA256 signature signed with the client_secret (private key) containing the concatenated ID and issued_at. Used to verify that the identity URL hasn’t changed since the server sent it.

token_type

token_type: string
  • Defined in flow/response/accessToken.ts:58

Value is Bearer for all responses that include an access token.
