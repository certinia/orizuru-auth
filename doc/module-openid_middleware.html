Module: openid/middleware

Express middlewares for user identitfication and authorisation. It is aimed at users of the Salesforce Identity Provider and designed to be used with the Orizuru framework.

(static) grantChecker(env) → {ExpressMiddleware}

Returns an express middleware that checks that an access token can be retrieved for the user specified on the request. Should be used in tandem with the tokenValidator middleware, and must be executed after that. This requires that a ConnectedApp is configured to pre authorise users and the user is authorised.

(static) tokenValidator(env) → {ExpressMiddleware}

Returns an express middleware that validates the OpenID Connect access token passed in an HTTP Authorization header and if successful sets the user object onto the request object.
